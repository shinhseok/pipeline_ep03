# 소품 턴어라운드 시트 생성 템플릿

## API content 배열 구조

```
THIS style — 이 이미지의 드로잉 스타일로 장면 전체를 그려줘:
<style_ref.png — 캐릭터 없는 앵커 이미지>

<프롬프트>
```

## 프롬프트 템플릿

```
THIS style의 드로잉 스타일로, {소품 설명}의
소품 턴어라운드 시트를 그려줘.

수평 한 줄로 4개 뷰를 나란히 배치:
1. 정면 (Front) — {정면에서 보이는 핵심 구조}
2. 측면 (Side) — {측면에서 보이는 핵심 구조}
3. 상면 (Top-down) — 위에서 내려다본 구조
4. 3/4뷰 — 약간 위에서 비스듬히, 입체감과 전체 구조

{소품 특징 상세}

모든 뷰에서:
- 동일한 소품 — 같은 형태, 같은 비율
- 높이 정렬
- 채색 없이 잉크 라인만

순백 배경. 텍스트, 라벨, 화살표, 주석은 넣지 마.
```

## 소품 뷰 구성 (캐릭터와 다름)

| 뷰 | 보여주는 것 |
|----|-----------|
| 정면 (Front) | 소품의 주요 면 — 가장 특징적인 앵글 |
| 측면 (Side) | 깊이, 두께, 메커니즘 구조 |
| 상면 (Top-down) | 위에서 본 평면 구조, 부품 배치 |
| 3/4뷰 | 입체감, 전체 구조 파악 |

## 예시 — spinning_wheel (물레)

```
THIS style의 드로잉 스타일로, 18세기 목제 물레(spinning wheel)의
소품 턴어라운드 시트를 그려줘.

수평 한 줄로 4개 뷰를 나란히 배치:
1. 정면 (Front) — 큰 구동 바퀴가 정면으로 보이게, 발 디딤판이 아래에
2. 측면 (Side) — 완전한 우측 측면, 바퀴의 두께와 방추 장치가 보이게
3. 상면 (Top-down) — 위에서 내려다본 모습, 바퀴 원형과 프레임 구조
4. 3/4뷰 — 약간 위에서 비스듬히 본 모습, 입체감과 전체 구조

물레 특징:
- 큰 구동 바퀴(10개 얇은 살)
- 발 디딤판(treadle)
- 수평 방추 장치
- 실패 거치대(distaff)
- 참나무 색조 목제 프레임

모든 뷰에서:
- 동일한 물레 — 같은 형태, 같은 비율
- 높이 정렬
- 채색 없이 잉크 라인만

순백 배경. 텍스트, 라벨, 화살표, 주석은 넣지 마.
```

## 설정
- model: gemini-3.1-flash-image-preview
- thinking_level: HIGH
- style ref: 채널 공통 앵커 이미지 (assets/reference/anchor/style_ref.png)

## 참고 — 캐릭터 시트와의 차이

| 항목 | 캐릭터 시트 | 소품 시트 |
|------|-----------|---------|
| ref | main.jpeg (체형+스타일) | style_ref.png (스타일만) |
| 뷰 구성 | 정면/3/4/측면/후면 | 정면/측면/상면/3/4 |
| 라벨 | 동일 ("형태만 따라") | 동일 ("형태만 따라") |
| 앞뒤 구분 | 필요 (엉덩이 라인) | 불필요 |
