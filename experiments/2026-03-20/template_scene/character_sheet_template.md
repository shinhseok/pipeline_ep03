# 캐릭터 턴어라운드 시트 생성 템플릿

## API content 배열 구조

```
THIS {base_character} — 이 캐릭터의 체형과 드로잉 스타일을 기반으로 그려줘:
<base_character 이미지 — 채널 앵커 캐릭터 (예: main.jpeg)>

<프롬프트>
```

## 프롬프트 템플릿

```
THIS {base_character}의 드로잉 스타일과 체형을 기반으로,
{캐릭터 설명} 캐릭터의 턴어라운드 시트를 그려줘.

수평 한 줄로 4개 뷰를 나란히 배치:
1. 정면 (Front) — 얼굴이 정면을 향해
2. 3/4뷰 (Front 3/4) — 약간 우측으로 돌린 자세
3. 측면 (Side) — 완전한 우측 프로필
4. 후면 (Back) — 완전한 뒷모습, 얼굴 안 보임, 엉덩이 라인 살짝

모든 뷰에서:
- THIS {base_character}와 동일한 콩 체형 — 같은 머리/몸 비율
- 똑바로 선 자세 (자연스러운 차렷)
- 머리/어깨/발 높이 정렬
{의상 디테일}

순백 배경. 텍스트, 라벨, 화살표, 주석은 넣지 마.
```

## 의상 디테일 작성 규칙

앞뒤 구분이 가능하도록 의상별 고유 디테일을 명시:

```
- 캐릭터 구분 포인트: {고유 의상/액세서리} ({워시 색} 유일한 색)
- 앞: {앞에서 보이는 특징}
- 뒤: {뒤에서 보이는 특징 — 앞과 반드시 다르게}
```

## 앞뒤 구분 규칙

콩 캐릭터는 앞뒤가 비슷하므로 반드시 구분 요소 필요:
- **앞**: 얼굴(점 눈) + 의상 앞면 디테일
- **뒤**: 얼굴 없음 + 엉덩이 라인 + 의상 뒷면 디테일 (매듭, 끈 등)

## 예시 — artisan

```
THIS main의 드로잉 스타일과 체형을 기반으로,
거친 리넨 작업복에 소매를 걷어올리고, 허리에 가죽 앞치마를 두르고,
머리에 부드러운 천 모자를 쓴 캐릭터의 턴어라운드 시트를 그려줘.

수평 한 줄로 4개 뷰를 나란히 배치:
1. 정면 (Front) — 얼굴이 정면을 향해
2. 3/4뷰 (Front 3/4) — 약간 우측으로 돌린 자세
3. 측면 (Side) — 완전한 우측 프로필
4. 후면 (Back) — 완전한 뒷모습, 얼굴 안 보임, 엉덩이 라인 살짝

모든 뷰에서:
- THIS main과 동일한 콩 체형 — 같은 머리/몸 비율
- 똑바로 선 자세 (자연스러운 차렷)
- 머리/어깨/발 높이 정렬
- 캐릭터 구분 포인트: 모자(warm-ochre 워시 유일한 색) + 앞치마
- 앞: 앞치마 매듭, 뒤: 앞치마 끈이 등 뒤로

순백 배경. 텍스트, 라벨, 화살표, 주석은 넣지 마.
```
