# Flow Prompt v3 패턴 — Visual Director Reference (has_human 3값 기반)

> v3 간소화: 구조적 태그 + 불필요 요소(스타일 묘사, 얼굴 규칙, REDRAW, 카운트, 장면 가드) 제거.
> 순수 한국어 서술형 4단락 구조 + THIS {name} 직접 참조.
> style_ref + 턴어라운드 시트가 외형/스타일을 담당 → flow_prompt는 구도/감정/채색만.

## 4단락 구조 (모든 패턴 공통)

| 단락 | 역할 | 내용 | YAML 소스 |
|------|------|------|-----------|
| **P1** | 과제 소개 | 교육 영상용 삽화 | `v3_task_intro` |
| **P2** | 배경 | 순백 배경 + 환경 힌트 | `v3_background_*` |
| **P3** | 구도/감정 | THIS {name} 크기+위치+포즈+감정 | creative_intent |
| **P4** | 채색 + 선 질감 | 워시 대상+색상 + **스케치 터치 강조** | `v3_color_rule_template` + sketch emphasis |

> **폐기**: 스타일 묘사(P2), 얼굴 규칙(P3), REDRAW(P3/P4), 카운트(P5), 장면 가드(P5)

## ref_images + THIS {name} 규칙

flow_prompt에서 **THIS {name}**으로 참조 (서수 참조 폐기).
`{name}`은 ref_images 배열의 파일명 stem.

**우선순위 배치:**
1. `costume_refs` → `characters/{RUN_ID}/{인물명}.jpeg` (우선)
2. `prop_refs` → ANCHOR Layer 1 묘사구 기반, `props/{RUN_ID}/{소품명}.jpeg` 경로
3. `character_prop` → `reference/` 직접 경로 (파일명만)

**has_human별 캐릭터 ref:**
| has_human | 캐릭터 참조 이미지 |
|-----------|------------------|
| `main` + `costume_refs: [변장명]` | `characters/{RUN_ID}/{변장명}.jpeg` |
| `main` + `costume_refs: []` (기본 해빛) | `assets/reference/style/main_turnaround.jpeg` |
| `anonym` | `assets/reference/style/character_reference.jpeg` (기본 형태 참조 — ref_images에 명시적 포함) |
| `none` | 캐릭터 ref 없음 |

> visual-director가 ref_images를 완전히 구성한다. style_ref, main_turnaround, character_reference 등
> 모든 참조를 명시적으로 포함해야 한다. generate_images.py는 추가/제거 없이 그대로 전달한다.
> **최대 5개 권장 (style_ref 포함)** — 이미지가 많으면 NB2 해석이 분산됨.
> ref 3개 이상 시 flow_prompt P1에 스타일 전이 방지 문구 포함:
> "여러 참조 이미지가 첨부되어 있어. 모든 요소에 동일한 THIS style의 드로잉 스타일을 유지해줘."

**경로**: `09_assets/reference/` 기준 상대 경로 (접두사 금지).

## 크기 표현 3중 체계

캐릭터와 소품의 크기를 반드시 3가지 방식으로 동시 표현한다:

```
전체 화면의 약 10%,         ← 수치 (프레임 점유율)
물레의 절반 크기로,          ← 상대비교 (다른 요소 대비)
마치 거대한 세계 속 하나의 점처럼   ← 은유 (감정적 스케일)
```

| 요소 | 수치 | 상대비교 | 은유 |
|------|------|---------|------|
| VAST 캐릭터 | 8-10% | 소품의 N분의 1 | "세계 속 하나의 점" |
| 일반 캐릭터 | 10-15% | 소품과의 비율 | 감정에 맞는 은유 |
| 강조 캐릭터 | 15-20% | 화면 중심 가까이 | "존재감이 느껴지는" |
| MCU (예외) | 20-30% | REVEAL/TENSION 절정 | "온 시선이 집중되는" |

## 제약 통합 패턴 3종

v3에서는 제약 사항을 서술문에 자연스럽게 통합한다. 별도 [MUST] 블록 없음.

### 1. 이유형 (P3/P4에 삽입)
```
캐릭터 얼굴은 코 없이, 작은 점 형태의 눈과 조용히 다문 입으로 집중하는 표정이야.
```

### 2. 카운트형 (P5 마지막 문장)
```
정확히 캐릭터 1명과 물레 1개만 그려줘.
```

### 3. 부정+이유형 (필요 시 P4/P5에 삽입)
```
텍스트, 글자, 화살표, 주석은 그리지 마 — 순수한 시각 작품만 만들어줘.
```

## 배경 밀도 등급별 P2 패턴 (has_human과 독립)

### L0~L2: P2에 배경 서술 통합

- L0: `v3_background_density.L0.prompt` (순백)
- L1: `v3_background_density.L1.prompt` (힌트 채움)
- L2: `v3_background_density.L2.prompt` (편린 채움) + "배경 선은 캐릭터 선보다 확실히 연하고 가늘게"

### L3 (sketch): P4에 단일 구조물 서술 추가

```
캐릭터 뒤로는 {구조물 묘사}이 얇고 연한 잉크 윤곽선으로만
그려져 있어 — 채색이나 세부 디테일 없이 구조만 암시하는 수준이야.
배경 선은 캐릭터 선보다 확실히 연하고 가늘게 그려줘.
```

### L4 (scene): P4에 다중 구조물 + 원근 서술

```
배경에는 {구조물들 묘사}이 얇고 연한 잉크 윤곽선으로 그려져 있어.
가까운 구조물은 약간 굵은 선으로, 먼 구조물은 더 가늘고 흐린 선으로
깊이감을 표현해줘. 채색이나 세부 디테일 없이 구조만 암시하는 수준이야.
배경 선은 캐릭터 선보다 반드시 연하고 가늘게.
```

### L5 (panorama): P4에 풍경 + 3단계 깊이감 서술

```
배경에는 {풍경 요소들 묘사}이 잉크 윤곽선으로 펼쳐져 있어.
원경은 극도로 가늘고 흐린 선으로, 중경은 약간 뚜렷한 선으로,
근경 구조물은 캐릭터 선보다는 연하지만 비교적 또렷한 선으로
3단계 깊이감을 표현해줘. 채색 없이 잉크 라인만으로 공간을 구성해.
```

> **L3~L5 공통**: 배경 요소에 채색 절대 금지 (잉크 라인만). 잉크 무게 계층: 캐릭터(1.0x) > 소품(0.7x) > 배경(등급별).
