# Pose Repertoire — 15개 포즈 아키타입 (Shot Composer Reference)

> shot-composer.md §6 (원칙 22) 참조 파일.
> 각 emotion_tag에 3개 아키타입 매핑. 6차원 + Line of Action 패키지.
> **동일 아키타입 2연속 금지** (원칙 11 참조).

## HUMOR 아키타입

### H1 — Sardonic Lean (냉소적 기울기)

| 차원 | 값 |
|------|-----|
| 기울기 | 한쪽으로 비스듬히 기울어짐 |
| 척추 곡선 | S-curve (느슨한) |
| 팔 간격 | 한 팔 가볍게 가리킴, 다른 팔 주머니 |
| 무게 중심 | 한 발에 체중 (비대칭) |
| 고개 방향 | 한쪽 갸우뚱 |
| 소품 상호작용 | 가볍게 가리킴 / 시선으로 지목 |
| **Line of Action** | S-curve |

> 한국어 묘사: "몸을 비스듬히 기울인 채 한 손으로 소품을 가볍게 가리키며 눈썹 한쪽을 치켜올리고 있다"

### H2 — Slapstick Spring (슬랩스틱 스프링)

| 차원 | 값 |
|------|-----|
| 기울기 | 뒤로 젖힘 / 위로 튀어오름 |
| 척추 곡선 | 역C-curve (상향) |
| 팔 간격 | 양팔 벌려 허우적 |
| 무게 중심 | 발끝 들림 / 점프 직전 |
| 고개 방향 | 뒤로 젖힘 |
| 소품 상호작용 | 소품에 압도당함 / 부딪힘 |
| **Line of Action** | 역C-curve |

> 한국어 묘사: "상체가 스프링처럼 튀어오르며 양팔을 벌려 허우적대는, 거대한 소품 앞에 압도당한 모습"

### H3 — Wry Slouch (시니컬한 축 처짐)

| 차원 | 값 |
|------|-----|
| 기울기 | 약간 앞으로 축 처짐 |
| 척추 곡선 | C-curve (하향, 약한) |
| 팔 간격 | 팔짱 또는 한 손 턱 괴기 |
| 무게 중심 | 양발 균등 (무기력) |
| 고개 방향 | 숙여서 내려다봄 |
| 소품 상호작용 | 내려다봄 / 무관심 |
| **Line of Action** | C-curve (약한 하향) |

> 한국어 묘사: "어깨가 축 늘어진 채 팔짱을 끼고 작은 소품을 내려다보며 반쪽 미소를 짓고 있다"

## REFLECTIVE 아키타입

### R1 — Contemplative S (사색의 S자)

| 차원 | 값 |
|------|-----|
| 기울기 | 약간 앞으로 기울어짐 |
| 척추 곡선 | S-curve |
| 팔 간격 | 뒤에서 깍지 / 한 손 턱 |
| 무게 중심 | 한 발에 체중 |

> 한국어 묘사: "몸이 부드러운 S자 곡선을 그리며 시선을 아래로 떨군 채, 먼 곳의 소품을 조용히 바라보고 있다"

### R2 — Melancholy Droop (우수의 처짐)

| 차원 | 값 |
|------|-----|
| 기울기 | 앞으로 깊이 숙여짐 |
| 척추 곡선 | C-curve (하향) |
| 팔 간격 | 소품 쥐고 있으나 힘 빠짐 |
| 무게 중심 | 양발 균등 (무거움) |
| 고개 방향 | 깊이 숙임 |
| 소품 상호작용 | 힘없이 쥐기 / 놓아주기 |
| **Line of Action** | C-curve (하향) |

> 한국어 묘사: "축 처진 어깨로 C자 곡선을 그리며 힘없이 소품을 쥐고 있는, 고개를 깊이 숙인 모습"

### R3 — Serene Stillness (고요한 직립)

| 차원 | 값 |
|------|-----|
| 기울기 | 거의 없음 (미세한 기울기) |
| 척추 곡선 | vertical (수직, 이완) |
| 팔 간격 | 양옆 자연스럽게 내림 |
| 무게 중심 | 양발 균등 (안정) |
| 고개 방향 | 정면 또는 눈 부드럽게 감김 |
| 소품 상호작용 | 조화로운 거리 |
| **Line of Action** | vertical (이완) |

> 한국어 묘사: "등을 곧게 세운 채 눈을 부드럽게 감고, 소품과 조화로운 거리를 두고 고요히 서 있다"

## AWE 아키타입

### A1 — Overwhelmed Recoil (압도된 뒤로 젖힘)

| 차원 | 값 |
|------|-----|
| 기울기 | 뒤로 크게 젖힘 |
| 척추 곡선 | 역C-curve |
| 팔 간격 | 팔 비율+20% 뻗어 올림 |

### A2 — Sublime Vertical (숭고한 수직)

|------|-----|

| 척추 곡선 | vertical (경직) |
| 팔 간격 | 양옆 내림 (최소) |

| **Line of Action** | vertical (상향 시선) |

> 한국어 묘사: "수직으로 꼿꼿이 선 채 고개만 하늘로 올리고, 광활한 여백 속 하나의 점처럼 서 있다"

### A3 — Wonder Reach (경탄의 뻗기)

| 차원 | 값 |
|------|-----|
| 기울기 | 앞으로 기울어짐 |
| 척추 곡선 | 역C-curve (약한) |

## 아키타입 선택 가이드

```
① emotion_tag 확인
② emotion_nuance 확인 (미기재 시 기본 뉘앙스)
③ 매핑: HUMOR → H1~H3, REFLECTIVE → R1~R3, AWE → A1~A3, REVEAL → V1~V3, TENSION → T1~T3
④ nuance → archetype 기본 매핑:
   sardonic→H1, slapstick→H2, wry→H3
   contemplative→R1, melancholy→R2, serene→R3
   overwhelming→A1, sublime→A2, wonder→A3
   eureka→V1, shock→V2, ironic→V3
   dread→T1, pressure→T2, defiant→T3
⑤ 서사적 필요 시 같은 tag 내 다른 아키타입 선택 가능
⑥ 원칙 11 확인: 이전 Shot과 동일 아키타입이면 반드시 교체
```
