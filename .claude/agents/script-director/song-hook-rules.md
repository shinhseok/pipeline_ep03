# Song Hook 가사 규칙 (§8.3)

> script-director.md의 §8.3 참조 파일. `02_planning`에서 `HOOK_TYPE: song` 선택 시에만 적용.

---

## 가사 작성 규칙

SECTION00 (Hook)의 나레이션을 **노래 가사**로 대체한다.

```
[SONG HOOK — {에피소드 주제}]

[Intro]
[Instrumental]

[Verse]
{에피소드 주제를 유머러스하게 요약 — 4~6줄, 50~70 음절}

[Hook]
{후크라인 1~2회 반복 — 20~30 음절}

[Outro]
{질문형 마무리 1줄 — 브릿지 연결용, 10~20 음절}
[End]
```

## 분량 기준

| 항목 | 기준 |
|------|------|
| 총 음절 수 | **80~120 음절** (±10% 허용) |
| 프레이즈 수 | 12프레이즈 (= 6줄 환산, 2프레이즈=1줄) |
| 프레이즈 단위 | **6~7음절** (긴장 변형 구간은 5 또는 9 허용) |
| 오프닝 Sound | **≤ 3초**, 리듬감 있는 소리 |
| 템포 | 중간 (~4 음절/초) |
| 길이 | 정확히 30초 |

## 가사 톤 & 설계 원칙

**10대 설계 원칙 (성공 패턴 기반)**:

1. **감각적 오프닝 (Sound Hook)**: 일상적이지만 집중을 유도하는 소리로 시작 (시계 초침, 키보드, 물소리 등). 가사 첫 줄 전에 `(소리 묘사)` 형태로 명시
2. **Show, Don't Tell**: 주제 키워드("AI", "산업혁명" 등)를 직접 말하지 않고 **장면 묘사**로 전달. 시청자가 스스로 연결하게 만든다
3. **시공간 전환**: 음악 안에서 현재 → 과거 (또는 반대) 시공간 이동. 가사 전반부와 후반부의 시대가 달라야 한다
4. **3단 감정 아크**: 호기심/긴장(오프닝) → 따뜻함/평화(중반) → 미세한 불안/여운(마무리). 평면적 감정 반복 금지
5. **마지막 2줄에 메시지 집중**: 앞 4줄은 장면, 마지막 2줄에서 핵심 인사이트를 자연스럽게 전달
6. **30초, 6줄 고정**: Verse 8프레이즈(4줄) + Outro 4프레이즈(2줄). 반복 구간(Hook x2)은 억지 강조 느낌 → 지양
7. **음악 프롬프트에 텍스처 비유 포함**: "연필 드로잉처럼", "수채화 번짐처럼" 등 시각적 비유로 사운드 질감 지정
8. **오프닝 Sound ≤ 3초 + 리듬감**: 길게 깔리는 앰비언트가 아닌, 짧고 리듬감 있는 소리 (타닥타닥, 똑딱 등). 3초 이내
9. **6-7음절 프레이징**: 한 호흡 단위 = 6~7음절. A-B 교차(7-6-7-6)로 노래하기 좋은 리듬. 2프레이즈 = 1줄 환산
10. **긴장 변형 (Tension Break)**: 마지막 전 구간에서 음절 패턴을 의도적으로 깨뜨림 (5→9 등 짧아지거나 늘어남) → 감정 전환점

**프레이징 패턴 예시:**
```
Verse:  7-6-7-6 | 6-7-6-7    (A-B 교차 리듬)
Outro:  5-9 | 7-7            (긴장 변형 → 해소)
```

## 작사-작곡 싱크 전략 (3종)

**A. Chord-Lyric Sync (화성-가사 정렬)**
- ♭VI 코드가 등장하는 감정 피크 지점에 **가장 감정적인 가사**를 배치
- 예: ♭VI 위치에 "나처럼" → 현재와 과거의 직접 연결
- MUSIC_PROMPT에 ♭VI 등장 시점과 가사 피크를 명시

**B. Syllable-Music Density Sync (음절-음악 밀도 동기화)**
- 빌드업(3~15초): 7-6-7-6 고른 음절 = 안정적 리듬
- 클라이맥스(15~25초): 6-7-6-7 약간의 변화 = 에너지 상승
- 아웃트로(25~30초): 5-9-7-7 불규칙 = 긴장 → 해소
- 음절 밀도와 악기 밀도가 같은 방향으로 움직여야 함

**C. Vowel-Emotion Sync (모음-감정 동기화)**
- 밝은 모음(아, 오): 따뜻함·희망 구간에 배치 (예: "햇살 든 **창**가에서")
- 어두운 모음(으, 우): 긴장·불안 구간에 배치 (예: "**끊**기던", "뭐가 남**을**까")
- 감정 전환점에서 모음 색깔도 전환

## 가사 표현력 강화 원칙

- **감각 구체화**: 추상 동작("열심히") → 감각 묘사("차가운 화면 위로")
- **온도 대비**: 차가운 장면(현재 디지털) ↔ 따뜻한 장면(과거 아날로그)으로 시공간 전환 강화
- **모티프 관통**: 핵심 이미지(예: "손")를 가사 전체에 걸쳐 변형·반복하여 순환 구조 형성
  - 첫 등장: "손끝"(현재) → 중간: "실 짜던 손"(과거) → 마지막: "내 손"(귀환)
- **♭VI 피크 직접 연결**: 감정 클라이맥스의 가사에 화자와 과거 인물을 직접 연결하는 표현 배치 ("나처럼")

**톤**:
- 편안하지만 매력적이고 중독적 — 강요하지 않는 자연스러운 전달
- 식상하지도 억지 웃음도 아닌 — 빨려들면서 메시지 전달 (Showing > Telling)
- 캐릭터 화법 유지하되, 과장·자기비하보다 서정적 관찰자 톤 우선

## 음악 프롬프트 병기 (필수)

Song Hook 가사 블록 아래에 `MUSIC_PROMPT:` 블록을 추가한다. 이 프롬프트는 STEP 06 audio-director가 suno_style 필드 작성 시 참조한다.
```
MUSIC_PROMPT: "{감각적 시작 소리(≤3초, 거의 무음→진입)}로 시작해서, {시공간 전환 묘사}.
체감 BPM {숫자}의 하프타임 {장르} 사운드. {메인 악기}가 선율을 이끌고 {서브 악기}가 받침.
조성: {키} 계열, {화성 진행 패턴}. 75% 지점에서 감정 클라이맥스.
{긴장 구간 묘사: 반음 하강/리듬 끊김 등}. {마무리 감쇠 묘사}.
질감: {시각적 텍스처 비유}. 화성 에너지 주도(멜로디+코드 중심, 타악 최소)."
```

**채널 기본 악기 구성**: 어쿠스틱 기타(메인) + 피아노(서브) + 첼로(긴장 구간)
**채널 기본 음악 구조 (30초 4단):**
| 구간 | 시간 | 음악 | 가사 |
|------|------|------|------|
| 인트로 | 0~3초 | Sound Hook → 에너지 진입 | (소리만) |
| 빌드업 | 3~15초 | 메인 선율 전개 | Verse 전반 (현재 장면) |
| 메인+클라이맥스 | 15~25초 | 75% 지점 감정 피크 | Verse 후반 (과거 전환) |
| 아웃트로 | 25~30초 | 반음 하강 + 감쇠 | Outro (긴장→여운) |

## [BRIDGE] 태그 — 노래 → 나레이션 전환 (필수)

Song Hook 직후, SECTION01 시작 전에 **브릿지 구간**을 명시한다:

```
[BRIDGE — Song → Narration]
[1] 후크라인 질문화: "♪ 그런데 정말 그럴까~ ♪"
[2] 침묵: (0.5~1초 PAUSE)
[3] 나레이션: "사실, 이건 우리 모두의 이야기입니다."
```

> ⚠️ [BRIDGE] 없이 노래에서 바로 나레이션으로 넘어가면 톤 전환 실패 → 시청 이탈 위험.
