# Broadcast Narration Polishing Rules (방송 나레이션 다듬기 원칙)

> STEP 03 최종본 편집 시 적용. 문체 7원칙 + 리듬 규칙과 함께 사용한다.

---

## 1. 귀로 듣는 글 — 기본 원칙

방송 원고는 되돌려 읽을 수 없다. **한 번에 이해되어야 한다.**

- 입으로 말하면서 쓴다 — 말이 걸리면 단어를 바꾼다
- 운율은 수필이 아니라 **시(詩)**에 가깝다
- 최종 검증은 반드시 **낭독 테스트**로 한다

---

## 2. 문장 구조 규칙

### 2-1. 문장 길이
| 기준 | 수치 |
|------|------|
| 최적 문장 길이 | **8~14어절** (20~40자) |
| 1분 나레이션 분량 | 약 280~300자 |
| 한 호흡 읽기 단위 | 7~12음절 |

### 2-2. 1문장 1정보
- 한 문장에 하나의 핵심 정보만 담는다
- 접속사 "~고, ~며"로 이어진 복합문 → 단문 2개로 분리
- 귀로 들을 때 정보가 2개 이상이면 뒤의 것을 놓침

### 2-3. 단문-장문 교차 리듬
- 단문만 이어지면 딱딱하고, 장문만 이어지면 지루하다
- **연속 단문 3개 초과 금지**, **연속 장문 2개 초과 금지**
- 음악처럼 장단을 섞어 리듬을 만든다

### 2-4. 주어 앞배치
- 주어(주체)를 문장 앞 1/3에 놓는다
- 주어가 늦게 나오면 귀로 듣는 시청자가 따라가지 못한다
```
❌ "겨울이 되면 공장을 찾는 사람이 많았는데요"
✅ "사람들은 겨울이면 공장을 찾았어요"
```

### 2-5. 능동태 + 현재형
- 수동태보다 능동태가 힘이 있다
- 과거 서술도 현재형으로 쓰면 현장감이 산다
```
❌ "아이들이 고용되었습니다"
✅ "공장주들은 아이들을 고용합니다"
```

---

## 3. 호흡과 침묵 설계

### 3-1. 호흡 단위
- 문장 안에서 자연스러운 **끊어읽기 지점**을 설계한다
- 쉼표(,) 위치 = 나레이터의 반 박자 쉼
- 긴 문장은 호흡점을 미리 설계한 뒤 쓴다

### 3-2. 침묵도 원고다
- 중요 정보 전후에 0.5~1초 쉼 → 집중력 배가
- 감정 전환점에서 침묵은 말보다 강력하다
- [PAUSE] 태그를 감정 설계의 일부로 다룬다

---

## 4. 전달력 강화

### 4-1. 화면-나레이션 분리 (중계방송 금지)
- 눈에 보이는 것을 그대로 반복하지 않는다
- 나레이션은 영상이 말하지 못하는 **맥락·의미·감정**을 담당
```
❌ "해빛가 책을 읽고 있습니다" (화면에 보이는 것 반복)
✅ "이 책 한 권이 250년을 건너뛰게 합니다" (의미 부여)
```

### 4-2. 미사여구 제거
- 형용사 대신 **팩트·숫자·구체적 이미지**로 대체
- 수식어 3개 이상 연속 금지 (문체 7원칙과 동일)

### 4-3. 쉬운 언어
- 전문 용어 → 친숙한 단어로 번역
- 구어의 군더더기(~인데요, ~거든요)는 절제

---

## 5. YouTube 리텐션 특화

### 5-1. 30~60초 마이크로 루프
- 영상을 하나의 긴 이야기가 아닌 **30~60초 단위의 소(小) 루프**로 분절
- 각 루프는 **호기심 갭(curiosity gap)**으로 끝나 다음 루프로 견인
- 매 30~45초마다 리텐션 훅: 질문, 예고, 소 반전

### 5-2. 감정 페이싱
- **긴장(Tension) → 해소(Release)** 패턴 반복
- 영상이 진행될수록 스테이크(stakes)를 점진적으로 높인다

### 5-3. 예측 가능성 제거
- 시청자가 다음 내용을 예상하면 이탈한다
- 순서 재배치, 패턴 인터럽트로 예측을 깬다

---

## 6. 실행 체크리스트

STEP 03 최종 편집 시 아래 항목을 순서대로 검증한다.

| # | 규칙 | 검증 방법 |
|---|------|-----------|
| 1 | 문장 길이 14어절 이하 | 길이 초과 문장 → 분리 |
| 2 | 주어가 문장 앞 1/3 | 주어 위치 확인 |
| 3 | 단문-장문 교차 | 연속 단문 3+, 연속 장문 2+ 금지 |
| 4 | 1문장 1정보 | "~고, ~며" 과다 사용 점검 |
| 5 | 미사여구 없음 | 형용사·부사 3+ 연속 없음 |
| 6 | 30~60초마다 리텐션 훅 | 타임라인 체크 |
| 7 | 침묵 구간 설계 | 감정 전환점에 PAUSE 존재 |
| 8 | 화면 중계식 반복 없음 | "~하고 있다" 단순 묘사 없음 |
| 9 | 낭독 테스트 | 소리 내어 읽어 걸림 없음 |
