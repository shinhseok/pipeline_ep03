# 리듬감 규칙 (§5)

> script-director.md의 §5 리듬 규칙 참조 파일.

---

## 규칙 1 — 7-어미 팔레트 회전

| 어미 | 정보 역학 기능 |
|------|--------------|
| `이에요/예요` | 기본 설명, 친근한 단정 |
| `거든요` | 이유·근거 제시 (뒤에 설명이 따라온다는 신호) |
| `잖아요` | 공유 경험 환기 (시청자도 알고 있다는 전제) |
| `네요` | 지금 막 발견하거나 실감하는 감각 |
| `군요/구나` | 사유의 여운, 혼자 중얼거리는 감탄 |
| `더라고요` | 과거 경험·발견의 생생한 보고 |
| `죠` | 시청자와 함께 확인하는 반문 |

적용: 5문장 블록 내 동일 어미 2회 초과 금지. `~습니다`는 5문장 블록 내 2회 이하.

---

## 규칙 2 — 3-1 리듬 사이클 (문장 길이)

| 등급 | 기준 | 예시 |
|------|------|------|
| S (짧음) | ~20자 이내 | "검색창이 먼저였거든요." |
| M (중간) | 21~45자 | "손이 폰으로 향하는 데 0.3초도 안 걸렸어요." |
| L (김) | 46~70자 | "우리가 무언가를 찾기 전에, 이미 검색창이 손 안에 들어와 있는 거잖아요." |

적용: S 3개 → L 1개 사이클 권장. 동일 길이 등급 4연속 금지.

---

## 규칙 3 — 문두 구조 다양성 (6유형 회전)

| 유형 | 예시 |
|------|------|
| 주어+은/는 | "AI는 …" / "우리는 …" |
| 시간부사 | "그 순간, …" / "어느 날, …" |
| 접속사 | "그렇다면 …" / "그런데, …" |
| 수사질문 | "혹시 …하신 적 있나요?" |
| 감탄·전환 | "사실은요, …" / "여기서 잠깐, …" |
| 명사구 단독 | "틀려볼 수 있는 시간." |

적용: 동일 문두 구조 3연속 금지.

---

## 규칙 4 — 완충어 제거

| 완충어 (금지) | 교체 형태 |
|-------------|---------|
| `~인데요` | `~이에요` / `~거든요` |
| `~것이에요` | `~에요` / `~죠` |
| `~수 있거든요` | `~거든요` / `~잖아요` |
| `~라고 할 수 있는데요` | `~이에요` / `~네요` |

---

## 규칙 5 — 마이크로 리스트 펀치

문단당 1회, 3항목 빠른 열거로 리듬 변화. 5항목 이상 나열 금지.
