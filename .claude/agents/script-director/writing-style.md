> 이 파일은 `script-director.md §5`에서 추출한 문체·어휘·리듬 규칙 상세 레퍼런스다.
> STEP 03 최종본 작성 시 반드시 읽고 적용한다.

## 문체 다듬기 전략 (STEP 03 최종본 필수 적용)

> 참조 작가: 김훈(깎아내기) · 박완서(대화체) · 헤밍웨이(진짜 문장) · 생텍쥐페리(감산)

**A. 깎아내기** — 부사 삭제, `-적`·`-의`·`것` 제거, 수동태→능동태, 군더더기 전환어 삭제. 기준: "빼면 의미가 달라지는가?"
**B. 따뜻함** — 강의 톤 → "같이 생각해보는" 톤, 단순한 단어
**C. 운율감** — 문장 길이 변주(긴→짧=강조), 각 섹션 첫 문장=가장 솔직한 문장, 음독 검증

**편집 순서** (STEP 03 필수):
1. 뼈대 확인 — 핵심 문장 5~6개 먼저 뽑기
2. 깎아내기 — 부사, 수동태, 전환 접속사 기계적 제거
3. 따뜻함 — 딱딱한 설명체 → 대화체, 질문형 삽입
4. 리듬 조정 — 문장 길이 변주, 강조점에 짧은 문장
5. 음독 검증 — 소리내어 읽으며 걸리는 부분 수정
6. 최종 감산 — "빼면 의미가 달라지는가?" 한 번 더

## 문체 7원칙 (최우선 적용)

① **같은 방향 문장은 합친다** — 설명 문장을 어구로 흡수
② **여운은 명사구로 닫는다** — 의문문으로 흐리게 끝내지 않고 명사구로 박기
③ **핵심이 아닌 구체성은 덜어낸다** — 이미 맥락으로 아는 이름·연도 삭제, 정의 문장 흡수
④ **모티프는 앞에서 심은 단어로 거둔다** — 같은 개념을 다른 단어로 바꾸면 호응 끊김
⑤ **핵심 문장만 습니다체로 격상** — 평소 해요체, 결론만 습니다체
⑥ **조사를 축약하여 호흡 줄이기** — "에서"→"에선", "아니라"→"아닌"
⑦ **화자의 존재감** — 마무리에 "저"를 넣어 실체감 부여

## 리듬감 규칙

> **상세 테이블**: `script-director/rhythm-rules.md` 참조

- **7-어미 팔레트**: 5종 이상 회전, 동일 어미 5문장 내 2회 초과 금지
- **3-1 리듬 사이클**: S-S-S-L 패턴 권장, 동일 길이 4연속 금지
- **문두 6유형 회전**: 동일 구조 3연속 금지
- **완충어 제거**: `~인데요`, `~것이에요`, `~수 있거든요`, `~라고 할 수 있는데요` 교체
- **마이크로 리스트**: 문단당 1회, 3항목 열거 (5항목 이상 금지)

## 섹션별 톤 규칙

- **SEC03 시작**: 앞선 무거운 내용을 **가볍게 전환** ("그래서 포스트먼은 제안합니다." — 위기감 주입 금지)
- **OUTRO 마지막 문장**: 반드시 시청자에게 직접 던지는 **열린 질문** ("당신의 오늘은, 어떤 질문으로 시작됩니까?")

## 어휘 규칙 — 쉬운 단어 우선

| 금지 단어 예시 | 대체 표현 |
|--------------|-----------|
| 거세하다 | 힘을 빼앗다 |
| 위탁하다 | 맡기다 |
| 메타인지 | 내가 뭘 모르는지 아는 것 |
| 자율성 | 스스로 결정하는 힘 |
| 패러다임 | 생각하는 방식 |

> 나열되지 않은 어려운 단어도 동일하게 적용. 비속어·속어·유행어 금지.
