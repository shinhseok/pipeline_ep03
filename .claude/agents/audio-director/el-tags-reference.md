# ElevenLabs Tag Reference

Tags are inserted at sentence start in `[BRACKETS]` format.

## Emotion Tags
```
[SAD] [ANGRY] [HAPPILY] [SORROWFUL] [EXCITED] [NERVOUS]
[FRUSTRATED] [TIRED] [CALM] [CHEERFULLY] [FLATLY] [DEADPAN]
[PLAYFULLY] [ANNOYED] [FLUSTERED] [CASUAL]
```

## Narrative Tone Tags
```
[DRAMATIC TONE] [WISTFUL] [SERIOUS TONE] [MATTER-OF-FACT]
[CONVERSATIONAL TONE] [SARCASTIC TONE] [AWE] [LIGHTHEARTED] [REFLECTIVE]
```

## Delivery/Pace Tags
```
[WHISPER] [RUSHED] [DRAWN OUT] [PAUSES] [HESITATES]
[STAMMERS] [INTERRUPTING]
```

## Nonverbal Tags
```
[LAUGHS] [LAUGHS SOFTLY] [SIGHS] [GASPS] [CLEARS THROAT]
```

## Punctuation Pacing
| Symbol | Effect |
|--------|--------|
| `.` | Full sentence pause |
| `;` | Short breath pause |
| `...` | Hesitation / lingering |
| `---` | Emphasis or cut-off |
