# Song Hook 성공 패턴 분석 — CH02 예시 기반

## 성공한 Song Hook (CH02) 해부

### 구조적 특징
- **정확히 30초**, 6줄
- **감각적 오프닝**: 시계 초침 소리 "똑딱 똑딱" — 일상적이지만 주의를 잡는 소리
- **시공간 전환**: 현대 알람 → 1760년대 영국 시골 → 기계음 암시
- **3단 감정 아크**: 긴장(알람) → 평화(햇살/첼로) → 불안(기계음)
- **메시지 자연 삽입**: 마지막 2줄에서 핵심 메시지 전달 ("기계가 가져온 새로운 시간의 리듬 / 우리의 월요일은 그렇게 사라졌죠")

### 음악 프롬프트 설계
- BPM 68 (느린 템포)
- 악기: 어쿠스틱 기타 + 따뜻한 첼로
- 텍스처 비유: "연필 드로잉처럼 거칠지만 따뜻한"
- 전환: 차가운 알람 → 평화 → 미세한 금속 기계음

### 왜 효과적이었나 (사용자 피드백)
1. **편안하지만 매력적이고 중독적** — 강요하지 않는 톤
2. **식상하지도 억지 웃음도 아닌** — 과장 없는 자연스러운 전달
3. **빨려들면서 메시지 전달** — Showing > Telling
4. **초침 소리** — 관심과 집중 유도

## CH02E1 현재 Song Hook 문제점

```
또 세상이 바뀐대요 / 이번엔 AI래요 / 내 일자리 없어진대 / 어쩌냐고요
```

| 문제 | 설명 |
|------|------|
| Telling 과잉 | "세상이 바뀐대요", "AI래요" — 직접적으로 말해버림 |
| 감각적 오프닝 없음 | 시작부터 가사 — 주의 끌기 장치 없음 |
| 시공간 전환 없음 | 음악 안에서 시대 이동이 없음 |
| 감정 아크 평면적 | 불안→불안→불안 — 3단 전환 없음 |
| 반복 구간 기계적 | "이 노래 200년 전에도 불렀대요" x2 — 강조가 아니라 뻔함 |

## 도출된 Song Hook 설계 원칙

1. **감각적 오프닝 (Sound Hook)**: 일상적이지만 집중을 유도하는 소리로 시작 (시계, 타자기, 물소리 등)
2. **Show, Don't Tell**: 상황을 그려서 메시지를 전달 — "AI래요"가 아닌 장면 묘사
3. **시공간 전환**: 음악 안에서 현재→과거 (또는 반대) 이동
4. **3단 감정 아크**: 긴장/호기심 → 평화/따뜻함 → 미세한 불안/여운
5. **마지막 2줄에 메시지**: 앞 4줄은 장면, 마지막에 인사이트
6. **음악 프롬프트에 텍스처 비유**: 시각적 비유로 사운드 질감 지정
7. **30초, 6줄**: 분량 고정
