# 2026-03-19 작업 기록

## 1. [directing_test 브랜치] 스틸 시퀀스 연출 전략 수립 + 파이프라인 반영

### 배경 및 논의

CH02E2 run002의 76개 Shot Record를 전수 분석한 결과, 현재 연출의 구조적 한계를 발견:

1. **나레이션-장면 중복 (65%)**: 대부분의 shot이 나레이션을 시각적으로 반복 (삽화 수준). 나레이션과 장면이 각각 다른 역할을 해야 시너지가 발생함.
2. **AWE 부재**: 76개 중 2개만 AWE 태그. 산업혁명/AI라는 압도적 주제를 다루면서 "거대한 것 앞 작은 존재" 구도가 거의 없음.
3. **캐릭터 크기 단조**: 8-15% 프레이밍에 40개 이상 집중. 25-35% 중간 크기 (결단/탐색 순간) 및 3% 이하 극소 크기 (시스템에 삼켜짐) 부재.

이어서 두들 애니메이션 고유 기법과 스틸 이미지 시퀀스의 고유 강점을 리서치:
- Scott McCloud "Understanding Comics" — 클로저(Closure), 6종 패널 전환, 거터(Gutter)
- Chris Marker "La Jetée" — 스틸 시퀀스 속 예외적 움직임의 파괴력
- Sempé 일러스트 — 발견의 여정(Eye Travel), 여백의 서사적 활용
- Ken Burns 다큐멘터리 — Dwell Time, 시간 주권
- Cartier-Bresson — 결정적 순간(Decisive Moment)

**핵심 결론**: Hook은 영상 체이닝을 유지하고, SECTION01~OUTRO 스틸 구간에서는 "개별 이미지 퀄리티"가 아닌 "연속된 이미지 사이의 관계"가 연출의 핵심.

### 수립된 전략: "장면으로 말하라"

**5원칙:**
1. **클로저 설계** — 두 shot 사이에서 시청자가 무엇을 상상하는지 의도적으로 설계
2. **발견 경로 설계** — 시청자의 눈이 그림 안에서 여행하는 순서를 설계 (하강/상승/수평/중심→주변/주변→중심)
3. **시간=감정 + Visual Lead** — duration_est를 시각적 무게로도 결정. 이미지-나레이션 타이밍 분리 (visual_lead 필드)
4. **결정적 순간** — 모든 shot은 "가장 팽팽한 찰나"의 freeze frame
5. **Counterpoint** — 나레이션이 사실을 말하면 장면은 체감을 전달. 같은 것을 말하면 중복 (삽화), 다른 것을 말하면 시너지 (연출)

**3기법:**
1. **컷 충격 (Scale Shock)** — 인접 shot 간 크기 3배+ 급변. 에피소드 3~5회 전략 배치
2. **여백 아크 (Whitespace Arc)** — 섹션 단위 여백 방향성 설계 (ANCHOR 레벨). 여백의 변화 자체가 서사
3. **구도 반복 (Mirror Composition)** — 같은 구도+다른 오브젝트로 시대 간 대비. 2~3회 사용

**Visual Lead (이미지-나레이션 타이밍 분리):**
- Scale Shock/AWE shot에서 이미지가 나레이션보다 2~3초 먼저 등장
- 시청자가 먼저 그림을 발견한 후 나레이션이 해설
- 수동 편집 가이드 (visual_lead 필드로 편집자에게 의도 전달)

### 기술적 판단 기록

- **iv_prompt 강화는 HOOK에만 해당**: SECTION01~OUTRO는 스틸 이미지이므로 iv_prompt의 변환/과정 묘사는 범용 규칙으로 만들지 않음
- **visual_lead 필드**: 수동 편집 워크플로우에 맞춰 "편집자 가이드" 역할. 자동화 시 영상 편집 스크립트에서 직접 사용 가능
- **선 품질(line_character)**: 현재 NB2 모델이 선 품질 차이를 얼마나 반영하는지는 검증 필요. 서술로 포함하되 효과는 다음 프로젝트에서 측정
- **Whitespace Arc**: PHASE A (narration_map) 단계에서 설계해야 Shot 배치 전체에 반영됨. PHASE B에서 개별 shot에 적용하면 늦음
