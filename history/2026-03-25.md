## 5. script-director VISUAL CUE 규칙 충돌 해소

### 문제
script-director 에이전트 내부에 규칙 충돌 존재:
- `output-format.md` (1줄): "VISUAL CUE는 대본에 포함하지 않는다"
- `script-director.md` 본문 (30건+): VISUAL CUE 삽입 규칙 상세 기술 (밀도, 형식, 감정 태그, HUMOR RATIO 등)
- 에이전트 description: "[VISUAL CUE] annotations" 명시

| `.claude/agents/script-director.md` description | "[VISUAL CUE] annotations" 제거, "Outputs pure narration text only — no VISUAL CUE" 추가 |

## 11. generate_images.py has_human 파싱 누락 수정 + 코드블록 래퍼 금지 규칙

### 문제
shot05, shot11에서 해빛 캐릭터가 그려지지 않음. `has_human: main`이 정상적으로 들어가 있는데도 `main_turnaround.jpeg`가 자동 첨부되지 않음.

### 근본 원인
`generate_images.py`의 `parse_yaml_shot_records()` 함수가 `has_human` 필드를 파싱하지 않았음. `shot.get("has_human")`이 항상 `None` → `"none"`으로 폴백 → 자동 첨부 조건(main일 때만) 스킵.

### 수정
| 파일 | 변경 |
|------|------|
| `generate_images.py` `parse_yaml_shot_records()` | 일반 Shot 분기 + Video Hook 분기 양쪽에 `has_human` 파싱 로직 추가 (`m_human = re.search(r"^has_human:\s*(\S+)", block, re.MULTILINE)`) |
| `.claude/agents/visual-director.md` Delta Output Format | 코드블록 래퍼(` ```yaml `) 금지 규칙 명시. YAML front matter(`---`)만 사용. |
| `.claude/agents/audio-director.md` YAML Body | 동일 — 코드블록 래퍼 금지 규칙 명시 |

### 검증
수정 후 shot05, shot11 재생성 → "채널 턴어라운드 자동 첨부: main_turnaround.jpeg" 로그 확인 → 캐릭터 포함 이미지 생성 성공.

## 14. Hook 체이닝 규칙 전면 개편 — 캐릭터 앵커 + 드론 카메라

### 문제
Hook Kinetic Transition 영상이 엉망. KF 이미지 간 시각적 연결이 끊김.

### 근본 원인
1. video_prompt가 이야기/시나리오를 서술하지, 촬영 지시(물리적 동작)를 기술하지 않음
2. KF 간 카메라 위치/구도가 급변
3. 캐릭터가 프레임에서 사라지거나 크기/각도가 급변

### 해결: 3대 원칙 수립
1. 캐릭터 = 앵커: 처음부터 끝까지 프레임 안에 존재
2. 카메라 = 드론: 캐릭터를 따라가고, 회전하고, 가감속
3. 배경 = 흘러가는 것: 프레임 밖으로 사라지거나 자연스럽게 전환

- shot-composer/execution-flow.md: Kinetic Transition 3대 원칙 + creative_intent 태그 구조 변경
